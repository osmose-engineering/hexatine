Glass Sponge Structures Visualizer
==================================

Files
-----
- glass_sponge_algorithms.py
    Parametric (schematic) geometry generators + Plotly figures for
    key motifs in glass sea sponge skeletal architectures.

- Glass_Sponge_Structures_Visualizer.ipynb
    Notebook "application" UI (Gradio + Plotly) to explore the motifs.

Usage
-----
1) Put both files in the same folder.
2) Open the notebook and run all cells.
3) Use the tabs to explore:
   - Macro skeleton (lattice + helical ridges + sieve plate + anchor tuft)
   - Lattice-only view
   - Spicule lamella cross-sections
   - Nano-scale silica nanosphere rings
   - Composite strut cross-sections
   - Anchor tuft
   - Cruciform spicule element

Dependencies
------------
- numpy
- plotly
- gradio

If needed:
    pip install numpy plotly gradio
